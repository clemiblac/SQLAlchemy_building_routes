Welcome to the query Home Page


In this project we have a database containing precipitation,temperature and station data. The date range is from 2010-01-01 to 2017-08-23. I use SQL Alchemy to allow the user to query the data they want from the database. Using the available routes, you can copy and paste the urls to view the data. Example: "myweather/api/v1.0/precipitation" to return the home page, you can hit the back button or come back to myweather

Below are the list of all available routes:
/api/v1.0/precipitation
/api/v1.0/stations
/api/v1.0/tobs

To view the data for specific temperature date ranges you can use the guidance below:
Date Ranges:
Enter your desired start date or end date as shown -- /api/v1.0/YYYY-MM-DD
/api/v1.0/<start>
/api/v1.0/<start>/<end>
